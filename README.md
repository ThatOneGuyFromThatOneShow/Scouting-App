# Scouting App

## Supported Browsers
### Desktop Browsers
* Chrome (And all Chromium browsers probably)

* Firefox

* Edge

### Mobile Browsers
* Chrome

* Safari


## Installation
Install CouchDB

Create a databasse named "scouting"

Install NPM

Run `npm install`

Run `npm run build`

Run `node server.js path/to/folder port` (Example `node server.js dist 80`)

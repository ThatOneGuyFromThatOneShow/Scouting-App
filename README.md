# scouting-app

## Supported Browsers
### Desktop Browsers
* Chrome (And all Chromium browsers probably)

* Firefox

* Edge


## Installation
Install CouchDB

Create a databasse named "scouting"

Install NPM

Run `npm install`

Run `npm run build`

Run `node server.js path/to/folder` (Example `node server/server.js dist 80`)


## Project setup
```
npm install
```
